## 一、项目概述

### 1.1 项目定位

**Gleamory** 是一个**个人网站首页**，作为一切项目的起点和展示平台。后续所有项目都会以卡片形式挂载在这个首页上。

### 1.2 项目目标

- 提供一个有设计感的项目展示入口
- 集中管理所有项目的展示信息
- 作为个人品牌的统一入口

### 1.3 访问方式

- **当前**: 纯静态页面
- **未来**: 可能扩展后端服务

## 二、功能需求

### 2.1 核心功能

#### 功能一：项目卡片展示

**功能描述**: 以卡片形式展示所有项目，点击可跳转到项目详情或在线地址

**布局规则**:
- 前 3 个项目以 **特色大卡片** 展示（全宽，封面图作为背景，文字叠加）
- 其余项目以 **网格卡片** 展示（3 列响应式）
- 没有封面图时显示渐变背景 + ✨ 占位

**卡片信息**:

| 字段 | 必填 | 说明 |
|:---|:---:|:---|
| 项目名称 | ✅ | 项目的显示名称 |
| 项目描述 | ✅ | 一句话描述项目 |
| 项目链接 | ✅ | 点击跳转的 URL |
| 项目状态 | ❌ | 开发中 / 已上线 / 已下线 |
| 项目标签 | ❌ | 多标签，如：在线网站、番剧、AI |
| 项目封面 | ❌ | 封面图，放在 `public/covers/` 目录 |
| 项目版本号 | ❌ | 如 v1.0.0 |
| 最近更新时间 | ❌ | 最后一次更新的日期 |

**交互行为**:
- 点击卡片 → 新窗口打开项目链接（不离开首页）
- 没有封面图 → 显示渐变占位背景
- 鼠标悬停 → 卡片上浮 + 品牌色光效

#### 功能二：时间线/动态

**功能描述**: 展示项目的更新动态，让用户了解最新进展

**动态内容**:
- 类型: 项目更新情况（如"xxx项目 v1.0 上线"）
- 格式: 纯文字，无配图
- 展示: 默认最近 10 条，可展开查看全部
- 排序: 最新的在最前面（数据倒序排列）

### 2.2 项目分类

**标签系统**:
- 支持多标签（每个项目可有多个标签）
- 标签可扩展（后续可添加新类别）
- 暂不做筛选功能（后续可扩展）

**预设标签**:
- 在线网站
- 番剧
- AI
- 桌面工具
- CLI 工具
- 设计资源
- 手机应用
- 后端服务

## 三、技术需求

### 3.1 技术栈

| 类别 | 技术选择 | 说明 |
|:---|:---|:---|
| 框架 | Vue 3 | Composition API + `<script setup lang="ts">` |
| 语言 | TypeScript 6.0 | 编译期类型安全，vue-tsc 检查 |
| UI 库 | Element Plus | 时间线组件，unplugin-element-plus 自动导入样式 |
| 构建工具 | Vite 5 | 开发热更新 + 生产构建 |
| 类型检查 | vue-tsc ^3.2 | `vue-tsc --noEmit` 集成到构建流程 |
| 代码规范 | ESLint（扁平配置） | typescript-eslint + eslint-plugin-vue |
| 格式化 | Prettier | 无分号、单引号、printWidth 100 |
| CSS 框架 | Tailwind CSS | 原子化 CSS |
| 字体 | Source Han Serif CN | 本地 OTF 加载（3 个字重） |
| 数据存储 | JSON 文件 | 静态数据，构建时打包 |

### 3.2 响应式设计

**适配设备**:
- 桌面端（≥1024px）：3 列网格
- 平板端（768px - 1023px）：2 列网格
- 手机端（<768px）：单列

**布局要求**:
- 自动根据屏幕大小调整布局
- 优先保证移动端体验

### 3.3 多语言支持

- **当前**: 仅中文
- **未来**: 可扩展为中英文切换

```json
  ]
```

```json
    }
  ]
}
```

- **整体风格**: 优雅、干净、有设计感

## 九、设计规范（已确认）

### 9.1 页面标题

- **中文名**: 微光集
- **英文名**: Gleamory
- **展示形式**: `Gleamory 微光集`

### 9.2 品牌配色

| 色值 | CSS 变量 | 用途 |
|:---|:---|:---|
| `#F783AC` | `--primary-pink` | 主粉色（标题渐变、圆点、边框） |
| `#B490E4` | `--primary-purple` | 主紫色（渐变、标签、hover 交互） |
| `#E05A8A` | `--primary-dark` | 深粉色（hover 加深） |
| `#FFD700` | `--accent-gold` | 金色点缀 |
| `#2D2D2D` | `--text-primary` | 主要文字 |
| `#5A5A5A` | `--text-body` | 正文文字 |
| `#9A9A9A` | `--text-secondary` | 辅助文字 |
| `#FFFFFF` | `--bg-white` | 白色背景 |

### 9.3 字体规范

- **字体**: Source Han Serif CN（思源宋体）
- **加载方式**: 本地 OTF 文件，通过 `@font-face` 声明
- **字重映射**:
  - Bold (700): 页面主标题 h1
  - SemiBold (600): 分区标题 h2、卡片标题 h3
  - Medium (500): 正文、按钮、标签等 UI 元素

### 9.4 圆角规范

- 卡片: 12px
- 标签/按钮: 8px
- 状态徽章: full (圆形)

### 9.5 动画规范

- **滚动渐入**: opacity + translateY(20px)，0.6s ease-out
- **卡片悬停**: translateY(-4px)，0.35s cubic-bezier
- **时间线展开**: 0.25s ease-out（收起时禁用动画）
- **返回顶部**: 0.3s ease
